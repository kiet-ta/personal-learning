# Triage Label Vocabulary

Use these canonical roles for local markdown issues.

| Role | Label | Meaning |
|---|---|---|
| Needs evaluation | `needs-triage` | A maintainer must decide whether and how to act. |
| Waiting on reporter | `needs-info` | The issue lacks required reproduction, context, scope, or acceptance criteria. |
| Agent-ready | `ready-for-agent` | Fully specified and safe for an AFK agent to implement. |
| Human-ready | `ready-for-human` | Valid work, but requires human judgment, access, or approval. |
| Won't fix | `wontfix` | The repo will not action this issue. |

Do not invent new label names unless the user changes this file.
